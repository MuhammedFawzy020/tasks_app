# Tasks Assignation Project

This project is a small app to assign tasks from admin to normal users and calculate numbers of tasks per users 
to help me watch performnace of every user.

## Features

* list All admin
* List All Users
* Create Tasks and assign it for users
* Watch statics of every users and their tasks

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

* Code Editor Like VS Code or Php storm 
* Git
* Xampp
* PHP Version 8.1
* Composer
* Mysql
* Laravel 10


### Installation

1. Clone the repository:
   * git clone https://github.com/MuhammedFawzy020/tasks_app.git

2. Navigate to the project directory:
   * cd tasks_app
3. Run One Command to install the composer and migrate database and seed DB
   * php artisan app:setup


## Runing The project

 * Open your browser and visit http://localhost:8000.
